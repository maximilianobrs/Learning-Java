# ☕ Variables en Java

Este repositorio contiene una colección de ejercicios prácticos enfocados en el aprendizaje y uso de variables en Java, utilizando distintos tipos de datos para representar información del mundo real.

## 📌 Descripción

Cada carpeta representa un ejercicio independiente donde se almacenan y muestran datos relacionados con diferentes contextos, como estudiantes, empleados, vehículos, videojuegos y cuentas bancarias.

El objetivo es desarrollar una comprensión sólida sobre la declaración, inicialización y utilización de variables en programas Java.

## 🧠 Conceptos aplicados

- Variables
- Tipos de datos primitivos
- Variables de referencia (`String`)
- Asignación de valores
- Concatenación de texto
- Salida por consola con `System.out.println()`

## 🚀 Cómo ejecutar

1. Abrir el proyecto en cualquier IDE compatible con Java.
2. Seleccionar el ejercicio que se desea ejecutar.
3. Ejecutar la clase `Main`.
4. Observar los resultados en la consola.

## 🎯 Objetivo del repositorio

Fortalecer los fundamentos de programación en Java mediante ejercicios simples que permitan practicar el uso de variables y tipos de datos, estableciendo una base sólida para conceptos posteriores como:

- Operadores
- Estructuras condicionales
- Bucles
- Métodos
- Programación Orientada a Objetos
